### 1.1 安装 OpenClaw


### 1.2 配置 AI 模型（推荐用国产模型，便宜好用）

| 模型 | 用途 | 月费估算 |
|------|------|---------|
| DeepSeek V3（免费/极低） | 选题、简单文案 | 0-20 元 |
| GLM-5（智谱） | 复杂文案、高质量输出 | 100-200 元 |
| Qwen-Max（通义） | 长文分析、数据报告 | 100-150 元 |

**省钱秘诀：** 简单任务用 DeepSeek V3（几乎免费），只有需要高质量输出时才切到 GLM-5。

```bash
# 配置模型（以智谱 GLM-5 为例）
openclaw config set model.primary "zhipu-glm-52/GLM-5-Turbo"
openclaw config set model.fallback "deepseek-v4-flash/deepseek-v4-flash"
```

### 1.3 准备 Chrome 浏览器（安全关键步骤 ⚠️）

**核心原则：使用你日常使用的 Chrome，不启动新的浏览器实例。**

```
1. 确保你的 Windows 电脑已安装 Chrome 浏览器
2. 先正常打开 Chrome，登录你的小红书账号
3. 关闭 Chrome
4. 以远程调试模式重新启动 Chrome：

   右键 Chrome 快捷方式 → 属性 → 目标栏末尾追加：
   --remote-debugging-port=9222

   完整示例：
   "C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222

5. 启动后，Chrome 就会在 9222 端口等待 CDP 连接
```

> ⚠️ **为什么不直接用 Playwright 启动浏览器？**  
> 因为 Playwright 启动的浏览器自带 `navigator.webdriver=true` 标记，小红书风控一行 JS 就能识别。而连接你自己的 Chrome，指纹完全真实。

### 5.2 详细操作步骤

#### 第一步：让 AI 帮你选题

在 OpenClaw 中发送：

```
帮我分析今天【你的领域，如"AI工具"】的热门话题，给我3个选题建议。

要求：
1. 结合当前热点和用户痛点
2. 适合知识付费定位
3. 给出每个选题的标题草稿和预计互动率
```

#### 第二步：AI 写稿

确认选题后发送：

```
帮我写一篇小红书笔记，主题是【你选的题目】。

要求：
- 标题15-20字，带数字和emoji
- 正文800-1200字，短句分段
- 语气像朋友分享，不要AI感
- 结尾引导互动
- 给出8-10个标签
```

#### 第三步：人工审核（最重要！）

检查清单：
```
□ 标题是否吸引人？（能不能让人想点进来）
□ 开头前3句是否抓住注意力？
□ 有没有AI感的句子？（如"以下是一些"、"总的来说"）
□ 内容是否真的有价值？（不是水话）
□ 有没有绝对化用语？
□ 有没有违规内容？（医疗、金融等敏感领域要小心）
□ 标签是否精准？
□ 读完有没有想关注的冲动？
```

**如果不满意，让 AI 重写：**
```
这篇文案AI味太重，请重写。要求更像真人随手写的，口语化，不要太整齐。
```

#### 第四步：生成配图

```
帮我生成一张小红书风格的封面图，主题是【你的主题】。
风格：简洁大方，文字醒目，适合1080x1440尺寸。
```

#### 第五步：自动发布

```
帮我发布这篇小红书笔记：
标题：【你的标题】
正文：【你的正文】
图片路径：【AI生成的图片路径】
标签：【你的标签】
```

Skill 会自动打开你的 Chrome 浏览器，执行发布流程。

## 第六步：账号安全防护（持续执行）

### 6.1 发布频率安全阈值

```
✅ 安全：
   - 每天 1-2 篇（最安全）
   - 每天 3 篇（安全上限）
   - 发布间隔 >30 分钟

⚠️ 危险：
   - 每天 >5 篇
   - 发布间隔 <5 分钟
   - 凌晨 0-6 点发帖

❌ 必封：
   - 每天 >10 篇
   - 短时间内连续发布
   - 固定时间精确发布
```

### 6.2 设置发布保护规则

在 OpenClaw 中配置以下保护：

```yaml
# 发布频率限制
max_posts_per_day: 2
min_interval_minutes: 60
quiet_hours:
  start: "00:00"
  end: "06:00"

# 内容安全检查（每次发布前自动执行）
content_safety:
  check_absolutewords: true    # 绝对化用语检测
  check_ad_law: true           # 广告法违规词检测
  check_sensitive_content: true # 敏感内容检测
  check_ai_tone: true          # AI 腔调检测
  require_human_approval: true # 必须人工确认才能发布
```

### 6.3 账号健康检查清单（每周执行一次）

```
□ 账号是否被限流？（发一条笔记看曝光量是否正常）
□ 笔记互动率是否稳定？（点赞+评论+收藏 / 阅读 > 5%）
□ 是否有违规通知？
□ Cookie 是否过期？（超过7天建议重新登录）
□ 近期发布的内容质量是否下降？
```

## 风险免责与最终忠告

### ⚠️ 必须了解的风险

1. **小红书 2026年3月已发布 AI 托管打击公告**，纯无人值守模式会被封号
2. **OpenClaw Skill 可能存在安全风险**，建议安装前用虾壳安全扫描
3. **自动化运营有封号风险**，请勿用主号测试，先拿小号试跑
4. **知识付费内容需有真实专业积累**，不要卖水分内容
5. **站外引流（留微信号等）违规**，被发现会限流甚至封号

### ✅ 最安全的使用方式

```
AI 负责：选题分析 → 写草稿 → 生成配图 → 数据收集
人工负责：审核内容 → 确认发布 → 回复评论 → 策略调整
```

> **记住：AI 是你的效率工具，不是你的替代品。**  
> 把重复劳动交给 AI，把决策权留给自己。  
> 这才是安全、可持续、能赚钱的正确打开方式。
