

## 设置用户名与邮箱

```
git config --global user.name "zhuangjx"
git config --global user.email "623155143@qq.com"
```

## 设置远程仓库

```
# 添加远程仓库，默认名称是 origin
git remote add origin https://github.com/ha321ha/zhuangjx.git

# 查看是否添加成功
git remote -v

# 修改
git remote set-url origin https://github.com/新用户名/新仓库名.git
```
